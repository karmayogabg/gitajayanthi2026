# ॐ கீதா ஜயந்தி 2026 போட்டிகள் (Gita Jayanti 2026)

கீதா ஜயந்தி 2026 போட்டிகளுக்கான அதிகாரப்பூர்வ இணையதளம் (Website).

---

## 🚀 GitHub Pages-ல் முதன்முறை வெளியிட (First-Time Setup)

இணையதளத்தை ஆன்லைனில் மக்கள் பார்வைக்கு கொண்டு செல்ல கீழே உள்ள படிகளைப் பின்பற்றவும்:

### வழிமுறை 1: GitHub இணையதளத்தில் (எளிதான வழி)
1. **[GitHub](https://github.com/)** கணக்கில் லாகின் செய்து, ஒரு புதிய **Public Repository**-ஐ உருவாக்கவும். அதற்கு `gitajayanthi2026` என்று பெயரிடவும்.
2. உங்கள் கணினியில் உள்ள இந்த இரண்டு கோப்புகளையும் (Files) அப்படியே அங்கே அப்லோடு (Upload) செய்யவும்:
   * `index.html`
   * `gita_upadesha_banner_1784008925717.jpg`
3. அப்லோடு செய்தபின் **Commit changes** என்பதை கிளிக் செய்யவும்.

### வழிமுறை 2: Command Line / Terminal வழியாக
உங்கள் கணினியின் டெர்மினலில் இந்த கோப்புறையினுள் (Folder) சென்று பின்வரும் கட்டளைகளை இயக்கவும்:
```bash
# Git-ஐத் தொடங்க
git init

# கோப்புகளைச் சேர்க்க
git add .
git commit -m "Initial commit"

# உங்கள் GitHub அக்கவுண்ட்டுடன் இணைக்க (உங்களது GitHub URL-ஐ மாற்றவும்)
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/gitajayanthi2026.git
git push -u origin main
```

---

## 🌐 இணையதளத்தை ஆன்லைனில் காண்பிக்க (Enable GitHub Pages)

கோப்புகளை ஏற்றிய பிறகு, அதை இணையதளமாக மாற்ற:
1. உங்கள் GitHub Repository பக்கத்தில் உள்ள **Settings** (பல்சக்கர ஐகான்) பகுதிக்குச் செல்லவும்.
2. இடதுபுற மெனுவில் **Pages** என்பதைத் தேர்ந்தெடுக்கவும்.
3. **Build and deployment** -> **Source** என்பதில் **Deploy from a branch** என்பதைத் தேர்ந்தெடுக்கவும்.
4. **Branch** என்பதில் `main` (அல்லது `master`) என்பதைத் தேர்ந்தெடுத்து, கோப்புறையை `/ (root)` என வைத்து **Save** செய்யவும்.
5. 1 முதல் 2 நிமிடங்களில் உங்கள் இணையதளம் பின்வரும் முகவரியில் நேரலையாகிவிடும்:
   👉 `https://YOUR_USERNAME.github.io/gitajayanthi2026/`

## 🔄 எதிர்காலத்தில் மாற்றங்களைச் செய்து அப்டேட் செய்ய (Future Updates & Deployment)

எதிர்காலத்தில் இணையதள பக்கத்தில் ஏதேனும் மாற்றங்களைச் செய்தால் (எடுத்துக்காட்டாக, ஶ்லோகங்கள் அல்லது தகவல்களை மாற்றினால்), அதை ஆன்லைனில் உடனே அப்டேட் செய்ய இந்த 3 கட்டளைகளை மட்டும் உங்கள் டெர்மினலில் இயக்கினால் போதும் (GitHub Pages தானாகவே புதிய மாற்றங்களை 1 நிமிடத்தில் அப்டேட் செய்துவிடும்):

```bash
# 1. நீங்கள் செய்த மாற்றங்களை Git-ல் சேர்க்க
git add .

# 2. என்ன மாற்றம் செய்தீர்கள் என்ற குறிப்புடன் சேமிக்க
git commit -m "மாற்றங்கள் பற்றிய விவரம் (எ.கா. ஶ்லோகங்கள் அப்டேட்)"

# 3. மாற்றங்களை ஆன்லைனுக்கு அனுப்ப
git push origin main
```
